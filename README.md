# NEXUS Light Designer

Advanced AI-Powered Design System with Multi-Model Orchestration

## Overview

NEXUS Light Designer is a comprehensive AI-driven design system built with React, TypeScript, and cutting-edge AI APIs.

## Features

- Multi-Model AI Integration
- Advanced Image Generation
- Collaborative Design
- Database Integration
- Modern UI/UX

## Tech Stack

- React 18+ with TypeScript
- Vite for fast development
- Tailwind CSS for styling
- Supabase for backend
- Gemini API for AI image generation
- Freepik API for design assets

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
